# JobMatch Calculator
This application can be used to calculate how likely you are to get a job you have interviewed for. The estimate is based on machine learning and generated data from 500 interviews. 

## Installation and setup
Clone the repository from GitHub. In your terminal run the command:

    ```bash 
    git clone <https://github.com/GitHub_username/JobMatch-Calculator.git>
    ```

    Navigate to the correct directory by running the command:
    ```bash
    cd Prosjekt_Omny
    ```

## Run the application
Install Live Server extension if you haven't already.
Right-click in the html-file and choose "open with live server".

The application should now be running on your browser.
